About This Analysis
  • Matching algorithm: Probabilistic record linkage via Splink 4.0.16 (link_type="link_only")
  • Blocking rules: exact email, 6-char email prefix, exact phone, exact zipcode
  • Comparison signals: Levenshtein on email/phone/address, Jaro-Winkler on name, exact zipcode
  • Auto-merge threshold: ≥ 95% confidence — 0 false positives on the baseline dataset
  • Review-queue threshold: 75–95% confidence — flagged for human review
  • Hard-case stress test: 20 intentionally tricky pairs (name/address changes, email domain swaps) — 95.9% recall
